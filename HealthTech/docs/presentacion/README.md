# Presentación del Proyecto HealthTech

## Descripción

Esta carpeta contiene la presentación utilizada como material de apoyo para la comunicación y exposición del proyecto HealthTech.

La presentación resume los aspectos principales del sistema, incluyendo:

* Problema que busca resolver.
* Descripción general de la solución propuesta.
* Estado actual de desarrollo.
* Arquitectura del sistema.
* Tecnologías utilizadas.
* Próximas implementaciones y evolución prevista.

## Nota

El contenido de esta presentación corresponde a la etapa actual del proyecto y podrá ser actualizado durante futuras entregas.

La versión final servirá como soporte visual para la exposición y defensa del proyecto.
